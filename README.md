# SCT_DS_1
# Task 01 – Distribution Analysis using Tableau

## Objective
To visualize the distribution of categorical and continuous variables using bar charts and histograms.

## Tool Used
- Tableau

## Dataset
A course enrollment dataset containing course categories, number of students, and course fees.

## Visualizations
1. Bar Chart – Number of Students by Course Category
2. Histogram – Distribution of Course Fees

## Key Insights
- Business courses have the highest student enrollment.
- Language courses have relatively lower enrollment.
- Most course fees fall within a mid-range, with fewer high-cost courses.

## Outcome
This task demonstrates the correct use of bar charts for categorical data and histograms for continuous data.
